# Ultipa Documentation

Source for the Ultipa product documentation. The rendered docs are published at
**[ultipa.com/docs](https://ultipa.com/docs)**. The docs are organized into product sections, each a top-level directory.

## Contributing

Found a typo, broken link, or unclear section? Contributions are welcome:

- **Report an issue**: open a [GitHub issue](https://github.com/ultipa/ultipa-docs/issues)
  describing the page and the problem.
- **Submit a change**: fork, edit the relevant Markdown file, and open a pull request.
  Keep changes focused on a single page or topic where possible.

## Reporting Security Issues

For documentation errors (typos, broken links, incorrect descriptions), please use [GitHub issues](https://github.com/ultipa/ultipa-docs/issues).

If you have discovered a security vulnerability in the **Ultipa product itself** (database engine, drivers, CLI, or hosted services), please do **not** open a
public issue. Email **support@ultipa.com** with subject line `SECURITY: <short description>` and we will respond within five business days.
