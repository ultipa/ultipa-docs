# Introduction

Ultipa Transporter supports cross platform operations of importing data from file, database, data warehouse, distributed stream, etc. as well as exporting data as CSV files. It has two components **Importer** and **Exporter**:

- `ultipa-importer`:
  - Support creating graphset, schema and property in the Ultipa database
  - Support importing into multiple schemas
  - Support prefix processing for data IDs

- `ultipa-export`:
  - Supports exporting to CSV (separated by comma)
  - Supports exporting into multiple schemas

## Prerequisites

- data files, or the link of data source
- a command line terminal, such as:
  - Linux: [bash](https://www.gnu.org/software/bash), [zsh](https://www.zsh.org/), [tcsh](https://www.tcsh.org/)
  - MacOS X Terminal, iTerm
  - Windows: [PowerShell](https://learn.microsoft.com/en-us/powershell/)
- a version of [Ultipa Importer/Exporter](https://www.ultipa.com/download) compatible with your operating system
