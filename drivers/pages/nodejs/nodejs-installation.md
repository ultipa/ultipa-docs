# Node.js

## Installation

## Supported Node.js Versions

The Ultipa Node.js SDK requires Node.js version 12.22.12 or higher.

## Install the SDK with NPM

> Visit <a target = "_blank" href="https://www.npmjs.com/package/@ultipa-graph/ultipa-node-sdk?activeTab=versions">SDK Version</a> to find your desired version of SDK. If the file does not exist, please contact <a href="mailto:support@ultipa.com">Ultipa Support</a>.

Create a Node.js project and a TypeScript file, and then add the SDK to your project using the following command:

<p tit="TypeScript"></p>

```bash
npm i @ultipa-graph/ultipa-node-sdk@~4.5.0-s4.4
```
