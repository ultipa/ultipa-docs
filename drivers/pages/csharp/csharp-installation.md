# C#

## Installation

## Supported NET Framework Versions

The Ultipa C# SDK requires .NET version 7.0 or higher.

## Install the SDK 

### Use NuGet Package Manager in Visual Studio

1. Go to the menu: **Tools** -> **NuGet Package Manager** -> **Manage NuGet Packages for Solution**.
2. Click on the **Browse** tab.
3. Search for the package by name `UltipaSharp` and toggle the **Include prerelease** checkbox.
4. Choose the version you want and click **Install**.

### Use .NET CLI

> Visit <a target = "_blank" href="https://www.nuget.org/packages/UltipaSharp">UltipaSharp</a> to find your desired version of SDK. 

Navigate to your project directory in your terminal and add the SDK to your project using the following command:

```bash
dotnet add package UltipaSharp --version 1.0.8-s4.4
```
