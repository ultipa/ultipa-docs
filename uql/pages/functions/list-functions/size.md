# size()

Function `size()` calculates the length of the <i>list</i>, namely, the number of elements in the <i>list</i>.

Arguments：
- List \<list>

Returns：
- Length \<number>

## Common Usage

Example: Calculate the length of [1,2,3]


```js
return size([1,2,3])
```
<p tit="Result"></p>

```bash
3
```
