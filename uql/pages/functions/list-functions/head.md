# head()

Function `head()` returns the first element of the <i>list</i>, namely, the element with index `[0]`.

Arguments：
- List \<list>

Returns：
- First element \<string>

## Common Usage

Example: Calculate the first element of [1,2,3]


```js
return head([1,2,3])
```
<p tit="Result"></p>

```bash
1
```
