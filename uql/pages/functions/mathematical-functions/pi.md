# pi()

Function `pi()` returns the value of PI.

Arguments：
- N/A

Returns：
- PI \<number>

## Common Usage

Example: Return the value of PI
 

```js
return pi()
```
<p tit="Result"></p>

```bash
3.14159265358979
```

Example: Convert 123° to radian value
 

```js
return 123 * pi() / 180
```
<p tit="Result"></p>

```bash
2.14675497995303
```
