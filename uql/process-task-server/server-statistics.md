# Server Statistics

Returned table name: `_statistic`
<br>
Returned table header: `cpuUsage` | `memUsage` | `expiredDate` | `cpuCores` | `company` | `serverType` (the CPU usage, memory usage in MB, expiration date, number of CPU cores, company name and server type of the current Ultipa instance)

Syntax:
<p tit="Syntax"></p>

```uql
// To check various system-wide statistical data in the current Ultipa instance
stats()
```
