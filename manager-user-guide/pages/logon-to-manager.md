# Logon to Manager 

To begin, open Ultipa Manager in your browser. Log in using your credentials.

<center><img src="https://img.ultipa.cn/img/2024-01-10-15-57-12-login.jpg" /></center>

Click <b>Create account</b> to register if you don't have an account. Only a username and password are required.

## User Roles

Ultipa Manager comprises two user roles:

- <b>Administrator:</b> This role includes both the root user and users granted administrative privileges.
- <b>Regular User:</b> The default role assigned after registration.

## User List

Once logged in as an administrator, access the <b>User List</b> in the upper-right corner of the <b>Instances</b> page.

<center><img src="https://img.ultipa.cn/img/2024-01-10-15-59-25-userList.jpg" /></center>

Administrators can edit user roles, reset user passwords and delete users (excluding the root user).
