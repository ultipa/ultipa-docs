# Introducing Ultipa Manager

Ultipa Manager is a graph database management system (GDBMS) designed to help you fully manage your databases, query and modify data, and visualize graphs.

> This page covers the basics of Ultipa Manager. For the complete guide, refer to <a target="_blank" href="/docs/manager-user-guide">Ultipa Manager</a>.

## Deployment and Access

Ultipa Manager can be easily deployed on <a target="_blank" href="https://cloud.ultipa.com/">Ultipa Cloud</a>. For on-premises deployment, please contact us at <a target="_blank" href="mailto:support@ultipa.com">support@ultipa.com</a>.

After deploying Ultipa Manager, you can access it by entering the service address in your web browser. Ultipa Manager is a web-based application.

## Connecting to Databases

<center><img src="https://img.ultipa.cn/img/2025-01-09-18-29-54-instances.jpg"></center>

Ultipa Manager supports the management of multiple database instances. You can add a new instance by entering its host details and user credentials to establish a secure connection.

## Feature Overview

<center><img src="https://img.ultipa.cn/img/2025-01-13-11-37-00-home.jpg"></center>

Once connected to an instance, Ultipa Manager enables a wide range of operations, including:

### Managing Graphs

Easily manage all graphs within an instance. For each graph, you can modify schemas, adjust properties, and manage the data it contains.

### Running GQL Queries

Write and execute GQL queries to interact fully with your graphs. The integrated editor provides syntax highlighting and suggestions to enhance the query-writing experience.

### Visualizing Results

View query results in various formats, layouts, and styles. Use built-in tools like maps and timelines to explore and interact with your data more effectively.

### Data Science and Analytics

Perform advanced graph analytics directly by using Ultipa's extensive library of graph algorithms, supporting use cases such as centrality ranking, community detection, and more.

### Loading Data

The Loader module facilitates data ingestion from various sources into Ultipa. You can map source data to nodes and edges, and perform bulk operations in either serial or parallel mode for optimal performance.

### Developing Widgets

Build custom widgets and applications using JavaScript together with Ultipa Manager's APIs, with support for HTML and CSS to design and style the front end.

### Controlling Access

Manage users, privileges, and database policies with fine-grained control, ensuring enterprise-grade security and compliance across your graph databases.
