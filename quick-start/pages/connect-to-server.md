# Connect to Server

> This page demonstrates how to connect to an Ultipa server via Ultipa Manager.

## Obtain Ultipa Manager

For users who deploy Ultipa server <u>on-premises</u>:
- If Ultipa Manager is bundled with the server product you purchased, deploy Ultipa Manager on-premises as your own convenience
- If not, please contact sales personnel for an entitlement

For users who deploy Ultipa server <u>via Ultipa Cloud</u>:
- Connect to an exclusive Ultipa Manager from the server list in your Ultipa Cloud account

## Connect to Ultipa Server

Users of Ultipa Cloud can proceed to step-3 directly.

1. Login to Manager

Sign up and login to Ultipa Manager. Google Chrome browser is recommended.

<center><img src="https://img.ultipa.cn/img/2024-01-10-15-57-12-login.jpg" /></center>

2. Save server information

Add and save a server connection. Multiple connections can be saved if multiple servers are deployed.

<center><img src="https://img.ultipa.cn/img/2024-01-10-16-00-22-instances.jpg" /></center>

> Use ',' to separate multiple nodes of a server cluster.

3. Test and connect

A green strip on top of the connection card suggests a good connection. Click 'Open' to start operating the server with Ultipa Manager.

<center><img width=400 src="https://img.ultipa.cn/img/2024-01-10-16-01-30-open.jpg" /></center>

4. Switch connection

If multiple connections are saved, connection can be switched anytime.

<center><img width=200 src="https://img.ultipa.cn/img/2024-01-10-16-04-55-switch.jpg" /></center>
