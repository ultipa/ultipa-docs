# Unique Identifiers

## Node Identifiers

Each **node** has a built-in unique identifier `_id`.

| Property | Value Type | Description |
| -- | -- | -- |
| `_id` | `STRING` | Node unique identifier. Its value can be either auto-generated by system or manually assigned during insertion. |

## Edge Identifiers

Each **edge** has a unique identifier `_id` and two endpoint references.

| Property | Value Type | Description |
| -- | -- | -- |
| `_id` | `STRING` | Edge unique identifier. By default, edges receive a system-generated `_id` and custom values cannot be assigned. When the `EDGE_ID` feature is enabled on the graph, the value may be manually assigned during insertion or auto-generated as a UUID v4. See <a target="_blank" href="/docs/gql/graphs-with-edge-id">Graphs with Edge ID Support</a> for details. |
| `_from` | `STRING` | The `_id` of the source node. |
| `_to` | `STRING` | The `_id` of the destination node. |
