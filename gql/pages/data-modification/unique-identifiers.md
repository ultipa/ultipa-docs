# Unique Identifiers

## Node Identifiers

Each **node** has a built-in unique identifier `_id`.

| Property | Value Type | Description |
| -- | -- | -- |
| `_id` | `STRING` | A string-type unique identifier. Its value can be either automatically generated by the system or manually assigned during insertion. |

## Edge Identifiers

Each **edge** has a system-generated unique identifier `_id` and two endpoint references.

| Property | Value Type | Description |
| -- | -- | -- |
| `_id` | `STRING` | A system-generated unique identifier. Custom `_id` values are not supported for edges. |
| `_from` | `STRING` | The `_id` of the source node. |
| `_to` | `STRING` | The `_id` of the destination node. |
