# Quick Start

**Ultipa Cloud** (<a target="_blank" href="https://cloud.ultipa.com">cloud.ultipa.com</a>) is a fully-managed, pay-as-you-go graph database service that operates on public cloud platforms. It simplifies the process of managing and exploring your graph data, allowing you to concentrate on data insights instead of infrastructure management.

Ultipa Cloud currently offers two main products:

* **Ultipa Graph Database:** Described as a "Graph Powerhouse," this is the core database product.
* **Ultipa Manager:** A one-stop-shop platform for managing and interacting with your databases. It offers a highly visualized and intuitive interface for data management and migration, querying and analytics, visualization, etc.

A **free trial** is now available.

## Login or Register

To access Ultipa Cloud, visit <a target="_blank" href="https://cloud.ultipa.com">cloud.ultipa.com</a>. You can either sign in to an existing account or sign up for a new one.

## Create an Instance

From the left-side navigation panel, navigate to **Ultipa Graph**. You can choose from three different instance types:

### Graph Powerhouse

This is the latest Ultipa graph database product (v5.x), with three tiers to choose from:

| <div table-width="17"></div> | Free Trial | Standard | Enterprise |
| -- | -- | -- | -- |
| **Configurations** | Fixed | Customizable | Fully customizable |
| **Manager** | **Manager (Lite)** included | **Manager (Lite)** included | **Manager** included |
| **Graph Size** | 2 graphs (each up to 1 million nodes and 1 million edges) | Unlimited | Unlimited |
| **Pricing** | Free | Pay-as-you-go | Custom pricing |

**Important Note:** The free trial instance will be automatically terminated when it expires. For this reason, please do not store any important data on it, as it will be permanently deleted.

**Manager (Lite)** is a limited-feature version designed to manage only the specific database instance it was created with. For a full-featured version, create a dedicated **Manager** instance.

Learn more about <a target="_blank" href="/docs/cloud-user-guide/graph-powerhouse">Graph Powerhouse</a>.

### Graph Blaze

This is the older Ultipa graph database product (v4.x). *This version is being deprecated and is not recommended for new instances.*

### Manager

This is the full-featured graph database management platform. A single **Manager** instance can be used to manage all your database instances.

Learn more about <a target="_blank" href="/docs/cloud-user-guide/manager">Manager</a>.

## Connect to an Instance

The simplest way to connect to a **Graph Powerhouse** instance is by clicking **Connect > Manager Lite** from the instance card.

You can also connect to **Graph Powerhouse** from your applications (Java, Python, Go, Node.js, or C#) or through other tools. <a target="_blank" href="/docs/cloud-user-guide/graph-powerhouse#Connect-to-an-Instance">Learn more</a>
