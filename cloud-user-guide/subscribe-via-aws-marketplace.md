# Subscribe via AWS Marketplace 

Ultipa Cloud offers upfront-commitment-free subscriptions via AWS Marketplace to provide greater convenience. You can subscribe to Ultipa Cloud services via AWS Marketplace by linking your AWS account to your Ultipa Account . 

## Prerequisites

- An AWS account with a valid payment method
   - If you do not have an AWS account, see <a href="https://repost.aws/knowledge-center/create-and-activate-aws-account" target="_blank"> how to create and activate a new AWS account</a>.
-  An Ultipa account
   - If you do not have an Ultipa account, <a href="https://www.ultipa.com" target="_blank">click here</a> to create one.

## Subscribe

1. Go to <a href="https://aws.amazon.com/marketplace" target="_blank">AWS Marketplace</a>, search "Ultipa Cloud" and click **Ultipa Cloud (Graph Database/SaaS)** from the result.

<center><img src="https://img.ultipa.cn/img/2024-07-15-14-00-48-cloud-in-aws-market.jpg"></center><br> 

2. Click **View purchase options**.

<center><img src="https://img.ultipa.cn/img/2024-07-15-14-03-01-view-purchase.jpg"></center><br>

3. Click **Subscribe**.

<center><img src="https://img.ultipa.cn/img/2024-07-15-14-07-10-subscribe.jpg"></center><br>

## Link AWS to Ultipa Cloud

1. Click **Set up your account** in the upper-right corner.

<center><img src="https://img.ultipa.cn/img/2024-08-01-16-11-42-set-up-your-account.jpg"></center><br>

2. Log in to Ultipa Cloud when you are directed to Ultipa Cloud.

<center><img src="https://img.ultipa.cn/img/2024-07-15-16-58-26-aws-ultipa-1.jpg"></center><br>

3. Click **LINK TO AWS**.

<center><img src="https://img.ultipa.cn/img/2024-07-15-14-54-02-link-to-aws-s2.jpg"></center><br>

<center><img src="https://img.ultipa.cn/img/2024-08-01-16-08-45-link-succ-aws.png"></center><br>

## Check AWS Billings

> After linking AWS account to Ultipa account, your payment method for Ultipa Cloud services is replaced by AWS billings. On **Billings** tab, check AWS billings as shown below. 
For more billings-related information, please read <a href="https://www.ultipa.com/policy/legal-notices/cloud-terms-of-use" target="_blank">Cloud Terms of Use</a>.
