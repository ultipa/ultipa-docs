# Subscribe via AWS Marketplace

Ultipa Cloud provides subscription access through AWS Marketplace without requiring any upfront commitment, offering greater convenience. You can subscribe to Ultipa Cloud services via AWS Marketplace by linking your AWS account to your Ultipa account.

## Prerequisites

- An AWS account with a valid payment method. If you do not have an AWS account, see <a href="https://repost.aws/knowledge-center/create-and-activate-aws-account" target="_blank"> how to create and activate a new AWS account</a>.
-  An Ultipa account. If you do not have an Ultipa account, <a href="https://www.ultipa.com" target="_blank">click here</a> to create one.

## Subscribe

1. Visit <a href="https://aws.amazon.com/marketplace/pp/prodview-sdmrcp4vtew5m" target="_blank">Ultipa Cloud</a> on AWS Marketplace and click **View purchase options**.

<center><img src="https://img.ultipa.cn/img/2025-05-22-17-36-28-marketplace.png"></center>

2. Scroll down and click **Subscribe**.

<center><img src="https://img.ultipa.cn/img/2025-05-22-18-04-23-subscribe.png"></center>

3. Wait for the subscription to complete. Account setup will be unavailable while the subscription is in progress.

<center><img src="https://img.ultipa.cn/img/2025-03-13-16-00-57-in-progress.png"></center>

4. Once the subscription is complete, proceed to set up your Ultipa account.

<center><img src="https://img.ultipa.cn/img/2025-03-13-16-01-01-purchase-complete.png"></center>

## Link AWS to Ultipa Cloud

1. Click **Set up your account** in the upper-right corner.

<center><img src="https://img.ultipa.cn/img/2024-12-30-17-08-26-set-up-account.jpg"></center>

2. Log in to Ultipa Cloud.

<center><img src="https://img.ultipa.cn/img/2024-07-15-16-58-26-aws-ultipa-1.jpg"></center>

3. Click **LINK TO AWS** to link your AWS account to your Ultipa account.

<center><img src="https://img.ultipa.cn/img/2024-07-15-14-54-02-link-to-aws-s2.jpg"></center>

## AWS Billings

After linking your AWS account to your Ultipa account, payments for Ultipa Cloud services will be processed through your **AWS billings**. You can view AWS billing details under **Billings**, as shown below.
